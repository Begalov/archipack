Importing dxf

How to fix common issues when importing dxf from cad packages ?

Precision issues, eg: files with georeference.

Problem : Entity are too far from scene center, so you likely to loose precision.
Solution : Open the file in a cad package, and center everything near 0,0 save (as new file) and re-import

Zoom issues

Problem : You cant see your lines, have issues when zooming.
Solutions : Right Side panel of 3d view (shortcut "n")->View->Clip increase End value Right Side panel of 3d view (shortcut "n")->Display->Grid Floor decrease Scale value
