How to duplicate objects ?

Window and Door draw tools does provide support to clone objects.
Select source object frame, press alt while clicking on "Draw" tool beside Window or Door, in Add Archipack -> Objects Press shift while creating objects to make independant copy, without shift pressed objects are linked.

Doors and Windows parameters are sored on the frame.
This means if you want duplicate you start by selecting the frame and then make your copy.
You'll end up with the frame being copied without other parts.
In order to recreate missing parts use the "refresh" button of object on the right panel.

All Archipack objects does support linked copy - ALT+D (when you change parameter on one all linked update according) regular copy (SHIFT+D) and "Copy to selected" for single parameters.

Doors an windows special notes

On top of linked objects you'll find an unlink button to break the link from other ones. You'll also find a delete button taking care of deleting all parts of compounds objects.
