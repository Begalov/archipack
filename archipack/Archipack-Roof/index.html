Allow "branched" roofs Triangular ends Dormer windows Roof Cutter to slice parts of roofs Up to 512 roof parts

Draft mode and quick edit for realtime edit even on complex roofs Preset system to share styles across projects Overrides for slope / width of each part * Slope 2d angle constraints for both ends and between parts
